# Prompt examples for Seedream 5.0 Lite

These prompts are starting points for testing Seedream 5.0 Lite through APIDot. Adjust brand names, subject details, source URLs, and safety rules before production use.

## 1. seedream-5.0-lite

A clean ecommerce hero image of a premium wireless speaker on a brushed metal desk, precise typography on a small label reading APIDot, realistic reflections, soft studio lighting, commercial product photography.

## 2. seedream-5.0-lite-edit

Keep the source product identity. Replace the background with a clean spring campaign setup, preserve realistic shadows, and make the final image suitable for an ecommerce hero asset.

## 3. custom size object

A structured product infographic with readable labels, clean iconography, and balanced spacing.

## Production prompt notes

- Store prompt versions together with task identifiers, selected model, owner, and request payload.
- Keep source media URLs reachable long enough for APIDot to fetch them.
- Be explicit about subject, composition, output constraints, quality tier, and source media.
- Do not log prompts or source media URLs if they may contain private customer data.
